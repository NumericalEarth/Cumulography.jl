# Cumulography.jl

*The art of cloudmaking*

GPU-native online simulations and tracing-based visualization of clouds using 
[Breeze](https://github.com/NumericalEarth/Breeze.jl) and [Hikari](https://github.com/JuliaGraphics/Hikari.jl)
following https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2023MS003797
